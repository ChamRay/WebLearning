# 2、Vue-cli的安装

## 2.1脚手架项目的下载和使用

~~~
npm install -g @vue/cli  通过npm全局安装vue脚手架3/4
vue create 'vue-name'    使用脚手架创建一个新的项目
npm run serve            运行新创建的项目
~~~

## 2.2 项目结构

~~~
node_modules 依赖目录
public 公共文件目录
src  开发文件夹
 |- assets 
 |- componnets 非路由组件
 |- router 路由
 |- store vuex
 |- views 路由组件
 - App.vue 
 - main.js 应用入口
package.json  包文件
	||- name
	||- version
	||- script 项目的脚本
	||- dependence 依赖
	||- dev-dependence k
 - package-lock.json
  ||- rules:{
  	在其中配置禁用规则
  }
 - vue.config.js
 - babel.config.js
~~~



## 2.3 eslint的禁用

在main.js中配置

~~~js
//由于eslint语言检查太严格了，在main中禁用

import Vue from 'vue'
import App from './App.vue'

//局部禁用
eslint-disabled no-unused-vars

Vue.config.productionTip = false

new Vue({
  render:h=>h(App),
}).$mount('#app')
~~~

~~~js
//全局禁用：在package.js文件同级目录下命名一个vue.config.js文件

//写在即想要的配置的东西去覆盖系统自带的配置
module.exports = {
  entry:'',
  outPut:{
    
  },
  loader:{
    
  },
  modules:{
    rules:[]
  },
  plugins:[],
  mode:''
  
  //禁用eslint
  lintOnSave:false
}
~~~
